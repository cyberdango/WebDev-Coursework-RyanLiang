In Unit 4, I learned more advanced HTML features such as attributes, lists, and embedding images into webpages. I learned how ordered lists, unordered lists, and nested lists can help organize website content clearly. I also learned the importance of accessibility in web design and how developers can make websites easier for everyone to use. One challenge was understanding the difference between relative and absolute image paths when adding images to a webpage. I improved my understanding by practicing image embedding exercises and testing different file paths. This unit helped me understand how structure, organization, and accessibility improve both the appearance and usability of a website.
